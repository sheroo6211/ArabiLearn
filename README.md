# ArabiLearn 🌙

Learn Arabic the natural way — real phrases, real conversations.

## What is ArabiLearn?
A free Arabic learning app for complete beginners and intermediate learners. Teaches the Arabic that people actually use every day, not textbook Arabic.

## Features
- 200+ real phrases across 8 categories
- Audio pronunciation for every phrase
- 4 practice modes with quizzes
- XP system, streaks and achievements
- Free to start — Premium unlock available
